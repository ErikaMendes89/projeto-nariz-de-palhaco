# Aplicativo de Filtro em Tempo Real usando a Rede Neural PoseNet

## Descrição
Este projeto é um aplicativo web simples que utiliza a biblioteca PoseNet do TensorFlow.js para aplicar um filtro de nariz de palhaço em tempo real. A aplicação captura a imagem da webcam do usuário e utiliza a rede neural para detectar a posição do nariz, sobrepondo a imagem do nariz de palhaço.

## Tecnologias Utilizadas
<ul>
<li>HTML5</li>
<li>CSS3</li>
<li>JavaScript</li>
<li>p5.js</li>
<li>ml5.js</li>
<li>Bootstrap</li>
</ul>

## Funcionalidades
<ul>
<li>Captura de vídeo em tempo real utilizando a webcam.</li>
<li>Detecção da posição do nariz utilizando a PoseNet.</li>
<li>Aplicação de um filtro de nariz de palhaço na posição detectada.</li>
<li>Botão para tirar uma foto com o filtro aplicado.</li>
</ul>

## Como Usar

<ol>
<li>Clone o repositório para sua máquina local.</li>
<li>Abra o arquivo index.html em um navegador de sua preferência.</li>
<li>Conceda permissão ao navegador para acessar a webcam.</li>
<li>O filtro será aplicado automaticamente em tempo real.</li>
<li>Clique no botão "Tirar Foto" para salvar uma imagem com o filtro aplicado.</li>
</ol>

## Estrutura do Projeto
<ul>
<li> `index.html`: Estrutura principal da aplicação.</li>
<li> `style.css`: Estilos personalizados para a aplicação.</li>
<li> `main.js`: Código JavaScript para captura de vídeo, detecção de poses e aplicação do filtro.</li>
</ul>
